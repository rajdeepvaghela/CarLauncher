# CarLauncher

CarLauncher is a simple, car-friendly Android launcher built with Kotlin. It provides a distraction-minimizing home experience for use while driving — large touch targets, fast app launch, and a clean interface.

## Features

- Clean, car-focused UI with large icons and readable text
- Quick-launch for frequently used apps (navigation, music, phone)
- Simple media and phone controls (if supported by device)
- Day/Night-friendly color scheme
- Built with Kotlin and Android SDK

> Note: Feature availability depends on the app implementation and device capabilities. See the source for exact behavior.

2. Open the project in Android Studio.
3. Let Android Studio download dependencies and sync Gradle.
